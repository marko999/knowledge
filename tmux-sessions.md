# new session
`tmux`

`tmux new`

`tmux new-session`

`tmux new -s sessionname`

# remove sessions
`tmux kill-ses`

`tmux kill-session -t sessionname`

# attach session
`tmux a`
`tmux att`
`tmux attach`
`tmux a -t sessionname`

# key bindings
`<prefix>$` - rename session
`<prefix>D` - detach session
`<prefix>)` - next session
`<prefix>(` - previous session
